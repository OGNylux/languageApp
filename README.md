# languageApp

A mobile Android app for language learning (Kotlin / Android Studio).

## Prerequisites
- Java JDK 11 or 17 installed
- Android Studio (at least Otter 2 Feature Drop) with Android SDK

## Install & Test
Open the project in Android Studio: choose `Open an existing Android Studio project` and select the project root.

### Test

Test the project in Android Studio via `Run 'app'`.

### Build
Build the project in Android Studio via `Build > Generate App Bundles or APKs > Generate APKs`.
